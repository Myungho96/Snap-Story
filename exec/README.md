## EC2

1. Docker 23.0.1 설치
2. Docker Compose 2.15.1 설치
3. git clone
    
    ```bash
    git clone https://lab.ssafy.com/s08-ai-image-sub2/S08P22A401.git
    ```
    
4. /S08P22A401/backend/default.conf → 도메인 수정
5. docker-compose up
    
    ```bash
    sudo docker compose up -d --build
    ```
